# Personal-Use Quickstart

This is the recommended first pass for a solo user or a small friend group.

1. Run `scan_project.py` on the installed game folder.
2. If the audit says `CAUTION`, extract only the visible text lane.
3. Validate the inventory.
4. Translate a 50-100 string sample first.
5. Reuse existing fonts if they already support Chinese.
6. Generate a patch/copy, not an in-place edit.
7. Do a static check and one quick in-game look if safe.

## What To Ignore In Pass 1

- Image text.
- Artistic logos.
- Deep runtime automation.
- Heavy archive repacking.
- Long bug-fix loops.

## Success Criteria

- The sample reads naturally.
- Placeholders survive.
- UI length looks okay.
- The user likes the tone.
- Spend stays tiny enough that continuing feels easy instead of intimidating.
